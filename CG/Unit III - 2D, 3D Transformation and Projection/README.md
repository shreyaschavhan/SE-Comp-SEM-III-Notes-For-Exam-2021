# Unit III : 2D, 3D Tranformation and Projection.

## 2D Tranformation:

* 2D Tranformation:
> * Repositioning and Resizing of an object is called as tranformation
> * Types of Tranformation:
> 	* Translation
> 	* Rotation
> 	* Scaling
> * Homogenious Co-odinates
> * Other Tranformation:
> 	* shearing 
> 	* Reflection
